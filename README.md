# Emu48 Skin – Droid48 Inspired

Custom skin for **Emu48** (HP 48/49/50 emulator for Android), featuring a **background image inspired by the Droid48 project** by **Arnaud Brochard**.

This skin aims to provide a modern look while staying visually consistent with the original Droid48 design, adapted to the Emu48 skin system.

---

## 📱 Compatible Emulator

- **Emu48 (Android)**  
  https://play.google.com/store/apps/details?id=org.emulator.forty.eight

---

## 🎨 Design Overview

- Background image inspired by the **Droid48** visual layout
- Clean and functional appearance
- Adapted for modern smartphone screens
- Focus on key readability and display clarity

> ⚠️ This skin is **not an exact copy** of Droid48.  
> It is an original adaptation inspired by the original project.
